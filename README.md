# Speedy File Organizer

Sort thousands of files in seconds. Completely offline.

## Features

- **Organize by file type** (extension and/or "kind") and creation date (year and/or month)
- **Handles thousands of files** in seconds
- **Completely offline** - your files never leave your device
- **No AI, no cloud, no tracking**
- **Exclude specific folders and files** from organization
- **Undo function** to reverse changes
- **Remove duplicates** during organization
- **BLAKE3 checks** to ensure data safety
- **Multi-language support**: English, Arabic, Chinese, Hindi, and Spanish

## Installation

### Linux

1. Download the appropriate `.tar.gz` file for your architecture
2. Extract: `tar -xzf [filename].tar.gz`
3. Make executable: `chmod +x filesorg`
4. Run: `./filesorg`

**Note:** Flatpak version coming soon for easier installation and automatic updates.

### macOS

1. Download `filesorg_apple.zip`
2. Extract the archive
3. Move the app to your Applications folder
4. Right-click and select "Open" on first launch (security requirement for unsigned apps)

### Windows

**Option 1: Microsoft Store**
1. Visit the [Microsoft Store page](https://apps.microsoft.com/detail/9nfvvt296q0n)
2. Click "Get" or "Install"
3. Launch from Start Menu

**Option 2: winget (Command Line)**
1. Open PowerShell or Command Prompt
2. Run: `winget install "Speedy File Organizer"`
3. Launch from Start Menu
