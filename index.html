  • Handles thousands of files in seconds
  • Completely offline - your files never leave your device
  • No AI, no cloud, no tracking
  • Exclude specific folders and files from organization

  • Available in five languages: English, Arabic, Chinese, Hindi, and Spanish

Installation Instructions

Linux

  1. Download the .tar.gz file
  2. Extract: tar -xzf file-organizer-linux.tar.gz
  3. Make executable: chmod +x file-organizer
  4. Run: ./file-organizer

Note: Flatpak version coming soon for easier installation and automatic updates.

macOS

  1. Download the .dmg file
  2. Open the downloaded file
  3. Drag File Organizer to your Applications folder
  4. Right-click and select "Open" on first launch (security requirement)

Windows

  1. Click the Microsoft Store button above
  2. Click "Get" or "Install" in the Store
  3. Launch from Start Menu
